Вставте ваш скрипт через кнопку «Текст» вгорі. Потім натисніть «Старт» або кнопку X / A на джойстику щоб почати прокрутку. ────────────────── Керування джойстиком: X / A → Старт та Стоп Стік або хрестовина вгору → перемотати назад Стік або хрестовина вниз → перемотати вперед R1 / RB → швидше L1 / LB → повільніше ────────────────── Жовта лінія посередині — місце читання. Кнопки A+ та A− змінюють розмір тексту. Гарної зйомки!

X / A — старт·стоп · стік — перемотка · R1/L1 — швидкість
